# 145. 二叉树的后序遍历

给定一个二叉树，返回它的 后序 遍历。

**示例:**

输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]

## 题解

### 1、递归

```python
class Solution:
    def postorderTraversal(self, root: TreeNode) -> List[int]:
        # 终止条件
        if not root:
            return []
        # 递归
        return self.postorderTraversal(root.left) + self.postorderTraversal(root.right) + [root.val]
```

### 2、迭代

```python
class Solution:
    def postorderTraversal(self, root: TreeNode) -> List[int]:
        stack = []
        node = root
        prev = None
        res = []
        # 开始遍历
        while stack or node:
            # 指针向左孩子深搜
            while node:
                stack.append(node)
                node = node.left
            # 当指针深搜到空节点时
            node = stack.pop()
            # 若此时节点的右子树未被遍历过 - 当前节点重新入栈，并开始遍历其右子树
            if node.right and node.right != prev:
                stack.append(node)
                node = node.right
            # 若此时节点无右子树 或 右子树已经被遍历过 - 根据后序遍历直接导入当前节点值
            # 并从栈中取出上一个节点
            else:
                res.append(node.val)
                prev = node
                node = None
        return res
```
